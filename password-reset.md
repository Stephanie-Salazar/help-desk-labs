# Password Reset Walkthrough

## Scenario
A user is unable to log into their account due to a forgotten password.

## Tools Used
- Windows
- Active Directory (lab environment)

## Steps Taken
1. Verified user identity
2. Located the user account in Active Directory
3. Reset the password and forced password change at next login
4. Confirmed user was able to log in successfully

## Outcome
User regained access to their account and confirmed the issue was resolved.

## What I Learned
- How password resets are handled in a domain environment
- Importance of verifying identity before making account changes
